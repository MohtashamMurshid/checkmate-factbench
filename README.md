# checkmate-factbench

A factbench library built with Bun.

## Installation

```bash
npm install checkmate-factbench
```

or

```bash
bun add checkmate-factbench
```

or

```bash
yarn add checkmate-factbench
```

## Usage

```typescript
import { /* your exports */ } from 'checkmate-factbench';
```

## Development

To install dependencies:

```bash
bun install
```

To build:

```bash
bun run build
```

To run:

```bash
bun run index.ts
```
